Basic Rules of Design

Good designs follow a basic set of rules. Although many consider there to be over twenty rules, I believe that good design can be boiled down to the following:

  1. Form Follows Function

    Design is determined by intention. Ask yourself the following questions before beginning a design:

    • Who is it for?
    • What is it intended to do?
    • What is it intended to convey?

  2. Simple Is Better

    Always remember, white space is magical. Use the following tips to make sure your design only includes the most important elements

    • Have a reason for every design element.
    • Use clear, meaningful, language.
    • Use images and copy that support the message. Leave them out if they don’t.
    • Choose colors that compliment the message instead of taking away from it.

  3. Communicate. Don't Decorate

    • Good designs should never overwhelm or confuse the user with unneeded information.
    • “I like it” is never a good reason to include an element in your design.

  4. Be Consistent

    • Your design should incorporate internal and external components and behaviors that are familiar to your audience.
    • If the user already understands the element, they don’t need to rethink and remember.

At the end of the day when you are considering design elements and a user experience. Think about these four prinicples: Usablility, Readability, Scalability, Consistency.
